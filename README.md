# HACKRPI_MINI


TO RUN WEBSITE:

Ensure eveything is downloaded on your system:
$ pip install Flask
$ pip install open ai

Prep enviorment:

$ export FLASK_APP=app
$ export FLASK_ENV=development


Run flask:
$ flask run

Open link in broswer:
Will look similar to: http://127.0.0.1:5000/

TO RUN CHATBOT:
Ensure eveything is downloaded on your system:
$ pip install open ai

Run Python:
$ python3 quanto.py

Chat away!
